shared entry and expiry-selection logic and only logs the common trigger event
when a short strike is breached or tested-side delta reaches `0.25`.

Ran `qc/spxw_7dte_baseline` in QuantConnect Cloud after pushing the project from

## Interpretation

The control result confirms the baseline is fragile enough to justify focused
repair testing. With an 89.41% net loss and 94.80% drawdown, the main value of
this run is establishing the severity and frequency of adverse post-trigger
paths that later variants need to improve.
